# 👰 每日老婆 插件使用文档

> 一个让群聊充满欢乐与“修罗场”的QQ机器人插件  
> 支持随机配对、许愿指定、强娶人妻、分手冷静等完整恋爱模拟系统

---

## ✨ 功能概览

| 指令        | 说明 | 每日限制 |
|-----------|------|----------|
| `今日老婆`    | 随机抽取一位群友成为伴侣 | 无 |
| `查询老婆`    | 查看当前伴侣信息 | 无 |
| `我要分手`    | 与当前伴侣分手 | 最多3次 |
| `/许愿 @用户` | 指定一位单身群友成为伴侣 | 最多3次 |
| `/强娶 @用户` | 牛走别人的伴侣 | 最多3次 |
| `/求婚 @用户` | 牛走别人的伴侣 | 最多3次 |
| `/老婆菜单`   | 显示本帮助菜单 | 无 |
| `/老婆关系图`  | 生成本群所有配对的关系图 | 无 |
| `/重置配对`   | **管理员**：清空本群所有配对数据 | - |
| `/重置冷静`   | **管理员**：清空本群所有冷静期记录 | - |

---

## 🎮 详细指令说明

### 1️⃣ `今日老婆`
- **效果**：系统从**未配对**且**不在你冷静期**的群成员中随机选取一人，与你结为伴侣。
- **注意**：如果你已经配对，再次执行会直接显示当前伴侣信息。

### 2️⃣ `查询老婆`
- **效果**：显示你当前伴侣的昵称、QQ号及头像。

### 3️⃣ `我要分手`
- **效果**：解除当前伴侣关系，并自动为**发起方**添加冷静期（默认48小时）。  
  冷静期内双方**无法再次配对或许愿**。
- **限制**：每天最多分手3次。

### 4️⃣ `/许愿 @用户`
- **效果**：直接指定一名**单身**群友成为伴侣。
- **限制**：  
  - 对方必须当前没有伴侣  
  - 双方不在冷静期内  
  - 每天最多许愿3次

### 5️⃣ `/强娶 @用户`
- **效果**：强行夺取**已有伴侣**的群友。  
  系统会自动解除对方与原配的配对（**不增加对方分手次数**，也不产生冷静期），然后与你建立新配对。
- **限制**：  
  - 自己必须当前没有伴侣  
  - 对方必须有伴侣  
  - 双方不在冷静期内  
  - 每天最多强娶3次

### 6️⃣ `/老婆关系图`
- **效果**：生成一张**圆形关系网络图**，显示本群所有已配对成员的**头像、昵称、QQ号**以及他们之间的连线。  
  图片自动发送到群内。

### 7️⃣ `/老婆菜单`
- **效果**：显示以上所有指令的简要说明。

## ⚙️ 配置说明（插件开发者 / 管理员）

插件支持通过 `_conf_schema.json` 在WebUI中修改配置项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `冷静期` | int | 48 | 分手冷静期时长（单位：小时）。冷静期内双方无法再次配对/许愿/强娶。 |
| `黑白名单` | list | `["all"]` | 群禁用/启用名单。`/`前缀代表禁用，`all`或`*`代表所有群。例如`all,/123456`表示允许所有群但禁用123456；`123456,234567`表示仅允许这两个群。 |

> 💡 修改配置后**无需重启机器人**，插件会自动读取新值。

## 🧠 核心机制详解

### 📅 每日重置
- 每天0点（以系统日期为准）自动**清空所有配对数据**，但**冷静期记录会保留**（仅清空配对，不会清空冷静期）。
- 所有用户的**分手次数、许愿次数、强娶次数**每日重置。

### ⏳ 冷静期
- 触发条件：执行 `我要分手` 后，**主动分手方**获得冷静期，对方**不产生冷静期**。
- 效果：冷静期内，**双方不能通过任何方式（随机/许愿/强娶）配对**。
- 时效：时长由配置项 `冷静期` 决定（默认48小时）。

### ♻️ 数据持久化
- 插件会自动将配对数据、冷静期记录保存到 `data/daily_wife.json`。
- 机器人重启后数据不会丢失。

### 👑 管理员命令
- `/重置配对`：清空当前群的所有配对信息（不清理冷静期）。
- `/重置冷静`：清空当前群的所有冷静期记录。

---

## ⚠️ 注意事项

1. **群聊专用**：所有指令仅支持群聊，私聊无效。
2. **艾特格式**：许愿和强娶必须通过 `@用户` 的方式指定对象（例如：`/许愿 @张三`）。
3. **冷静期检查**：无论配对、许愿还是强娶，都会检查双方是否处于冷静期。
4. **头像获取**：插件会尝试下载QQ头像，若失败会显示文字提示，不影响配对逻辑。
5. **机器人自身**：机器人和发送者本人不会出现在随机抽取列表中。

## 🐛 常见问题

| 问题 | 原因与解决 |
|------|------------|
| 发送`今日老婆`提示“找不到合适人选” | 群内所有人都已配对或处于冷静期，等待冷静期过期或使用管理员重置。 |
| 许愿/强娶后没有成功 | 检查对方是否单身/有伴侣，以及双方是否在冷静期内。 |
| 分手后立刻想复合 | 冷静期内无法操作，请等待时间结束或由管理员重置冷静期。 |
| 关系图生成失败 | 检查是否安装了PIL/Pillow库，或字体文件 `ldm.ttf` 缺失（不影响生成，仅字体回退）。 |
